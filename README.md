Dynamically Updatable Index for Indexed Text Search
---------------------------------------------------

How to Run:

1. Place the following files in a folder:

	main.cpp
	genome.cpp
	test.cpp
	utils.cpp
	data/input.txt
	Makefile

2. Run "make" on your terminal from the this folder

3. Run "./main < data/input.txt" from your terminal
	


Dependencies:
C++11
